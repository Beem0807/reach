  kubectl -n {{ .Release.Namespace }} get pods -l app.kubernetes.io/instance={{ .Release.Name }}

  kubectl -n {{ .Release.Namespace }} get lease {{ include "reach-agent.leaseName" . }}

there and run commands against the cluster with `reach exec`, e.g. kubectl get pods -A.
